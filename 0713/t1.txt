답변
이 코드는 리스트 컴프리헨션(List Comprehension, 리스트 내포)문법입니다. 기존 리스트에서 조건에 맞는 값만 골라 새로운 리스트를 간단하게 만들 때 사용합니다.
문법 형태
새로운_리스트 = [표현식 for 변수 in 반복가능한_객체 if 조건]
• 표현식: 새로운 리스트에 넣을 값
• for 변수 in 반복가능한_객체: 리스트를 하나씩 순회
• if 조건: 조건을 만족하는 값만 저장(생략 가능)


문제의 코드 분석
high_scores = [score for score in scores if score >= 80]
하나씩 보면,
• score→ 새로운 리스트에 저장할 값
• for score in scoresscores리스트를 하나씩 꺼내서 score에 저장
• if score >= 80→ 80점 이상인 경우에만 저장
즉,
scores 리스트를 하나씩 검사하여 80점 이상인 점수만 high_scores에 저장한다.


같은 기능을 일반 반복문으로 작성하면
high_scores = []              # 빈 리스트 생성

for score in scores:          # scores를 하나씩 확인
    if score >= 80:           # 80점 이상이면
        high_scores.append(score)   # 리스트에 추가
실행 결과는 두 코드 모두 같습니다.
[85, 92, 95, 88]

네, 있습니다. Pandas에서는 같은 결과를 여러 방식으로 구현할 수 있습니다. 이전 방법은 DataFrame의 행 인덱스를 학생명으로 지정하는 방식이었고, 아래는 학생 열을 추가하거나 NumPy 기능을 함께 사용하는 다른 방법입니다.
방법 2: 학생 열을 직접 추가하는 방식
import numpy as np
import pandas as pd

data = np.array([
    [85, 90, 88],
    [70, 80, 75],
    [95, 98, 100],
    [60, 72, 68]
])

# 2.1, 2.2 DataFrame 변환 및 열 이름 지정
df = pd.DataFrame(data, columns=['국어', '영어', '수학'])

# 학생 번호 열 추가
df.insert(0, '학생', ['학생1', '학생2', '학생3', '학생4'])

# 2.3 학생별 평균 추가
df['평균'] = df[['국어', '영어', '수학']].mean(axis=1)

# 2.4 평균이 가장 높은 학생 출력
max_student = df.loc[df['평균'].idxmax()]

print(df)
print("\n평균이 가장 높은 학생")
print(max_student)

핵심 부분 설명
• pd.DataFrame(data, columns=...)→ NumPy 배열을 Pandas DataFrame으로 변환하고 과목명을 지정한다.
• df.insert(0, '학생', [...])→ DataFrame 첫 번째 열에 학생 이름을 추가한다.
• df[['국어','영어','수학']].mean(axis=1)→ 세 과목 점수의 평균을 계산한다.
• df.loc[df['평균'].idxmax()]→ 평균 값이 가장 큰 행(학생)의 모든 정보를 가져온다.

이 방법은 실제 데이터 분석에서 학생 번호, 이름, ID 같은 식별자 열이 따로 있는 경우더 많이 사용하는 형태입니다. 이전 방법은 인덱스를 활용하는 간단한 형태이고, 이 방법은 데이터베이스 표 구조와 비슷해서 확장하기 쉽습니다.

• NumPy 배열은 CPU 기반의 일반적인 수치 계산에 주로 사용된다.
• TensorFlow Tensor는 딥러닝 모델 계산을 위해 GPU/TPU 연산을 지원한다.
• Tensor는 자동 미분과 머신러닝 모델 학습에 필요한 기능을 제공한다.
핵심 코드 설명
• tf.constant(array): NumPy 배열을 TensorFlow Tensor로 변환한다.
• tensor.shape: Tensor의 행과 열 크기를 확인한다.
• tensor.dtype: Tensor 내부 데이터 타입을 확인한다.
• tensor + 10: Tensor의 모든 원소에 같은 값을 더하는 연산을 수행한다.

5.1 Conv2D 역할
• Conv2D(합성곱 층)은 이미지의 특징을 추출하는 역할을 합니다.
• 필터(커널)를 이용하여 모서리, 선, 질감, 모양등의 특징을 학습합니다.
• 앞쪽 층에서는 단순한 특징을, 뒤쪽 층에서는 더 복잡한 특징을 추출합니다.


5.2 MaxPooling 역할
• MaxPooling은 특징 맵의 크기를 줄이는 역할을 합니다.
• 일정 영역(예: 2×2)에서 가장 큰 값만 선택하여 출력합니다.
• 이를 통해
• 연산량을 감소시키고,
• 중요한 특징을 유지하며,
• 과적합(Overfitting)을 줄이는 효과가 있습니다.


5.3 Flatten 역할
• Flatten은 2차원(또는 3차원) 특징 맵을 1차원 벡터로 변환하는 역할을 합니다.
• CNN에서 추출한 특징을 Dense(완전 연결층)에 입력할 수 있도록 변환합니다.
예를 들어,
7 × 7 × 64
를
3136
개의 1차원 데이터로 변환합니다.


5.4 Dense 역할
• Dense(완전 연결층)는 앞에서 추출한 특징을 이용하여 최종 결과를 예측하는 역할을 합니다.
• 모든 입력 노드와 모든 출력 노드가 연결되어 있습니다.
• Dense(128)은 특징을 학습하는 은닉층이고,
• Dense(10)은 10개의 클래스를 예측하는 출력층입니다.


5.5 Softmax를 사용하는 이유
• Softmax는 출력값을 0~1 사이의 확률로 변환합니다.
• 모든 클래스의 확률 합이 1이 되도록 만들어 줍니다.
• 따라서 여러 클래스 중 가장 확률이 높은 클래스를 최종 예측 결과로 선택할 수 있습니다.
예를 들어 출력이
[2.1, 0.5, 3.2]
였다면 Softmax를 적용하면
[0.24, 0.05, 0.71]
처럼 확률로 변환되어 3번째 클래스(71%)를 예측하게 됩니다.

답변
5.1 Conv2D 역할
Conv2D(Convolutional 2D)는 이미지에서 중요한 특징(Feature)을 추출하는 층입니다.
• 여러 개의 필터(Filter, Kernel)를 이미지 위에서 조금씩 이동시키며 연산합니다.
• 각 필터는 특정한 특징을 찾아냅니다.
• 선(Line)
• 모서리(Edge)
• 곡선(Curve)
• 질감(Texture)
• 물체의 형태(Shape)
학습이 진행될수록 필터의 값이 자동으로 변경되어 더 중요한 특징을 잘 추출하게 됩니다.
예를 들어 고양이 사진을 학습한다면
• 첫 번째 Conv2D → 눈, 귀의 경계선
• 두 번째 Conv2D → 얼굴 모양, 귀 모양
• 마지막에는 "고양이"라는 특징을 인식할 수 있게 됩니다.
Conv2D(32)는 32개의 필터를 사용하여 32개의 특징 맵(Feature Map)을 생성한다는 의미입니다.


5.2 MaxPooling 역할
MaxPooling은 특징 맵의 크기를 줄이는(다운샘플링) 층입니다.
예를 들어
1 4
3 2
2×2 영역에서 가장 큰 값인 4만 남깁니다.
즉,
1 4
3 2
↓
4
이렇게 중요한 정보만 유지합니다.
사용하는 이유
• 데이터 크기를 줄여 계산 속도를 높입니다.
• 메모리 사용량을 줄입니다.
• 작은 위치 변화에도 같은 특징을 인식할 수 있습니다.
• 과적합(Overfitting)을 줄이는 데 도움이 됩니다.


5.3 Flatten 역할
Conv2D와 MaxPooling을 거치면 데이터는 다음과 같은 형태가 됩니다.
7 × 7 × 64
이 데이터는 3차원입니다.
하지만 Dense층은 1차원 데이터만 입력받을 수 있습니다.
그래서 Flatten이
7 × 7 × 64
↓
3136
으로 한 줄로 펼쳐 줍니다.
즉,
Flatten은 이미지의 특징 정보를 잃지 않으면서 1차원 벡터로 변환하는 역할을 합니다.


5.4 Dense 역할
Dense는 완전 연결층(Fully Connected Layer)입니다.
모든 뉴런이 이전 층의 모든 뉴런과 연결됩니다.
입력층
 ● ● ● ●

 ↓↓↓↓

Dense층
 ● ● ● ● ●
Dense층에서는 Conv2D가 추출한 특징을 이용하여
• 어떤 숫자인지
• 어떤 동물인지
• 어떤 물체인지
최종적으로 분류합니다.
예를 들어
Dense(128)
은 128개의 뉴런이 있는 은닉층입니다.
여기서 특징들을 조합하여 중요한 정보를 학습합니다.
마지막
Dense(10)
은
0
1
2
...
9
총 10개의 숫자를 분류하기 위한 출력층입니다.


5.5 Softmax를 사용하는 이유
Dense(10)의 출력은 처음에는 단순한 점수(Score)입니다.
예를 들어
[-2.3, 1.2, 4.5, 0.3, ...]
이 값만으로는 어느 클래스일 가능성이 높은지 직관적으로 알기 어렵습니다.
Softmax를 적용하면
[0.001,

처럼 0~1 사이의 확률로 변환됩니다.

Softmax를 사용하는 이유
• 출력값을 확률로 변환한다.
• 모든 클래스의 확률 합이 1이 된다.
• 가장 높은 확률을 가진 클래스를 최종 예측으로 선택할 수 있다.
• 다중 분류(Multi-class Classification) 문제에서 널리 사용된다.

      │
      ▼
MaxPooling     → 특징은 유지하면서 크기 축소

      │

이렇게 보면 Conv2D는 특징을 추출하고, MaxPooling은 중요한 특징만 남기며 크기를 줄이고, Flatten은 Dense층에 입력할 수 있도록 데이터를 1차원으로 변환하고, Dense와 Softmax는 최종 분류를 수행한다고 이해하면 됩니다.

역할
핵심 내용
Conv2D
이미지의 특징 추출
필터(Filter)를 이용해 선, 모서리, 질감, 모양 등의 특징을 학습한다. 층이 깊어질수록 더 복잡한 특징을 추출한다. Conv2D(32)는 32개의 필터를 사용한다.
MaxPooling
특징 맵 크기 축소
일정 영역에서 가장 큰 값만 선택하여 출력한다. 계산량 감소, 학습 속도 향상, 과적합 방지, 중요한 특징 유지에 도움이 된다.
Flatten
1차원 변환
Conv2D와 MaxPooling을 거친 2차원(또는 3차원) 데이터를 1차원 벡터로 변환하여 Dense층에 입력할 수 있도록 한다.
Dense
최종 분류(완전 연결층)
이전 층의 모든 뉴런과 연결되어 추출한 특징을 종합하여 결과를 예측한다. Dense(128)은 특징을 학습하는 은닉층, Dense(10)은 10개의 클래스를 출력하는 출력층이다.
Softmax
확률 계산
출력값을 0~1 사이의 확률로 변환하며, 모든 확률의 합이 1이 되도록 한다. 가장 높은 확률의 클래스를 최종 결과로 선택한다.

⬇

6.3 오류가 발생한 이유를 설명하시오.
• numbers는 파이썬의 리스트(list)자료형입니다.
• 리스트에는 .max()메서드가 존재하지 않습니다.
• 따라서 numbers.max()를 호출하면 AttributeError가 발생합니다.
• 리스트에서 최댓값을 구할 때는 **내장 함수 max()**를 사용해야 합니다.
• 반면 **NumPy 배열(numpy.ndarray)**은 .max()메서드를 제공하므로 사용할 수 있습니다.

Sequential로 모델을 만들고, Flatten으로 1차원 변환 후 Dense층으로 특징을 학습하고 최종 분류한다.
7.3 학습
model.fit(x_train, y_train, epochs=5)
학습 데이터를 이용하여 모델을 반복 학습시킨다.
7.4 평가
model.evaluate(x_test, y_test)
테스트 데이터로 모델의 손실값과 정확도를 평가한다.
7.5 예측
model.predict(x_test)
새로운 데이터의 클래스를 예측하며, 각 클래스의 확률을 출력하고 가장 높은 확률의 클래스를 최종 결과로 선택한다.

답변
1. 손실 함수 (Loss Function)
의미
• 모델의 예측값과 실제값의 차이(오차)를 계산하는 함수이다.
• 손실값이 작을수록모델의 성능이 좋은 것이다.
• 학습의 목표는 손실값을 최소화하는 것이다.
그림
입력 데이터
     │
     ▼
   AI 모델
     │
     ▼
예측값(8) ───────┐
                 │ 비교
실제값(10) ──────┘
        │
        ▼
  Loss = 오차 계산
대표적인 손실 함수
• MSE : 회귀 문제
• Cross Entropy : 분류 문제


2. 옵티마이저 (Optimizer)
의미
• 손실 함수의 값을 최소화하도록 가중치와 편향을 수정하는 알고리즘이다.
• 모델이 더 정확한 예측을 하도록 학습시킨다.
그림
현재 가중치
     │
     ▼
손실 계산
     │
     ▼
Optimizer
     │
가중치 수정
     │
     ▼
손실 감소
대표적인 Optimizer
• SGD
• Adam(가장 많이 사용)
• RMSprop


3. 활성 함수 (Activation Function)
의미
• 뉴런의 출력값을 변환하는 함수이다.
• 신경망에 비선형성(Non-linearity)을 추가하여 복잡한 문제를 학습할 수 있게 한다.
그림
입력
 │
 ▼
가중치 계산
 │
 ▼
Activation Function
 │
 ▼
출력
대표적인 활성 함수
• ReLU
• Sigmoid
• Tanh
• Softmax


4. Softmax
의미
• 출력값을 0~1 사이의 확률로 변환하는 활성 함수이다.
• 모든 클래스의 확률 합이 1이 된다.
• 다중 분류 문제의 출력층에서 사용한다.
그림
Dense 출력
[2.3, 1.2, 4.5]

      │
      ▼
Softmax

[0.09, 0.03, 0.88]

가장 큰 확률 → 최종 예측
특징
• 확률로 변환
• 확률의 합 = 1
• 가장 큰 확률의 클래스를 선택


5. ReLU
의미
• 가장 많이 사용하는 활성 함수이다.
• 음수는 0, 양수는 그대로 출력한다.
그림
입력 → 출력

-3 → 0
-1 → 0
 0 → 0
 2 → 2
 5 → 5
장점
• 계산이 빠르다.
• 기울기 소실(Vanishing Gradient) 문제를 줄인다.
• CNN, DNN에서 가장 많이 사용된다.


6. Epoch
의미
• 전체 학습 데이터를 한 번 모두 학습한 횟수를 의미한다.
그림
학습 데이터
[전체]

Epoch 1
 ↓
전체 데이터 1번 학습

Epoch 2
 ↓
전체 데이터 다시 1번 학습

Epoch 3
 ↓
전체 데이터 다시 1번 학습
예를 들어
• Epoch = 10
• 전체 데이터를 10번 반복 학습한다.

Softmax
출력값을 확률(0~1)로 변환하며 확률의 합이 1이 되도록 한다. 다중 분류의 출력층에서 사용한다.
ReLU
음수는 0, 양수는 그대로 출력하는 활성 함수로 계산이 빠르고 학습 성능이 우수하여 가장 많이 사용된다.
Epoch
전체 학습 데이터를 한 번 모두 학습한 횟수이며, Epoch가 증가할수록 반복 학습이 이루어진다.
